# Criterios de Evaluación

La evaluación será binaria:

* **Entregó:** Cumple con los requisitos mínimos de la actividad.
* **No entregó:** La actividad no fue entregada o no funciona.

## Acreditación de la Práctica

Para acreditar la práctica el estudiante deberá:

* Entregar el código fuente de cada actividad.
* Demostrar la ejecución de los programas durante la revisión.
* Entregar un archivo comprimido con todos los scripts desarrollados.
* Incluir una breve descripción de cada solución implementada.

## Evidencias

Cada actividad deberá incluir:

* Código fuente.
* Captura de pantalla de ejecución o demostración en laboratorio.
* Comentarios básicos dentro del código indicando autor y propósito del script.
